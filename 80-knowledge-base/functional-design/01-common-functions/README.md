# Common Functions

โฟลเดอร์นี้เก็บเอกสาร Functional Design สำหรับ **Common Functions** ที่ใช้ร่วมกันทั้งระบบ

## Naming Convention
- Prefix: `COM`
- Format: `com-[NNN]-[short-name].md`
- ตัวอย่าง: `com-001-login.md`, `com-003-menu-navigation.md`

## ตัวอย่าง Common Functions
- Login / Authentication
- User Profile
- Menu / Navigation
- Home Dashboard
- Notification Center
- User Preferences / Settings

## สำคัญ
- **Menu Navigation (COM-003) ต้อง define ก่อนเสมอ** เพราะเป็นตัวกำหนดโครงสร้างของระบบ
- ทุก function ต้องลงทะเบียนใน Function Index ก่อนสร้างเอกสาร
