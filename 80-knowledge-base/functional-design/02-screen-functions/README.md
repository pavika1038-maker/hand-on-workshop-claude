# Screen Functions

โฟลเดอร์นี้เก็บเอกสาร Functional Design สำหรับ **Screen Functions** (หน้าจอที่ผู้ใช้โต้ตอบ)

## Naming Convention
- Prefix: `SCR`
- Format: `scr-[NNN]-[short-name].md`
- ตัวอย่าง: `scr-001-create-request.md`, `scr-002-approval-screen.md`

## ตัวอย่าง Screen Functions
- Create / Edit Form
- Approval Worklist
- Search / Filter Screen
- Detail / History View
- Dashboard Screen

## สำคัญ
- ทุก Screen Function ต้องอ้างอิง Menu Navigation (COM-003) ว่าเข้าถึงจากเมนูไหน
- ทุก function ต้องลงทะเบียนใน Function Index ก่อนสร้างเอกสาร
