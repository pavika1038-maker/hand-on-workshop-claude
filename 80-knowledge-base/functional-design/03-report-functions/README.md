# Report Functions

โฟลเดอร์นี้เก็บเอกสาร Functional Design สำหรับ **Report Functions** (รายงานและ Dashboard)

## Naming Convention
- Prefix: `RPT`
- Format: `rpt-[NNN]-[short-name].md`
- ตัวอย่าง: `rpt-001-leave-summary.md`, `rpt-002-usage-dashboard.md`

## สำคัญ
- ทุก function ต้องลงทะเบียนใน Function Index ก่อนสร้างเอกสาร
