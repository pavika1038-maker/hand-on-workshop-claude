## 1. Introduction

### 1.1 Overview
เอกสารฉบับนี้อธิบายขั้นตอนการตรวจสอบรายงานการขายเพื่อบันทึกบัญชี โดยครอบคลุม:
- ใบกำกับภาษี
- ใบเพิ่มหนี้
- ใบลดหนี้  
ทั้ง **เงินสด / เครดิต / งานภายใน**  
จาก **ศูนย์บริการ, อะไหล่ และลูกหนี้อื่น**  
รวมถึงเอกสารที่ถูกยกเลิก

### 1.2 Business Background
ระบบปัจจุบันยังไม่มีรายงานที่รองรับการตรวจสอบรายได้เพื่อบันทึกบัญชีอย่างครบถ้วน  
สำนักงานใหญ่และสาขาจึงต้องการรายงานใหม่เพื่อตรวจสอบข้อมูลได้ชัดเจนและสะดวก

### 1.3 Purpose
1. ตรวจสอบรายละเอียดรายได้ของเอกสารแต่ละใบ  
2. ตรวจสอบยอดรวมขายสด / เครดิต / งานภายใน  
3. ตรวจสอบแหล่งที่มาของเอกสาร (แผนก)  
4. ตรวจสอบจำนวนเอกสารและยอดรวมระดับสาขาและทุกสาขา

---

## 2. Overview Scope

รายงาน **"การขายเพื่อการบันทึกบัญชี"**  
- ทำงานบนระบบ **Enterprise Application System**
- แสดงข้อมูลจากหลายเมนูเพื่อประมวลผลในรูปแบบรายงาน

> ⚠️ หมายเหตุด้าน Performance  
รายงานดึงข้อมูลจากหลาย Table ซึ่งอาจกระทบประสิทธิภาพขึ้นกับปริมาณข้อมูล  
ทั้งนี้ **นอก Scope โครงการ** และต้องเสนอขออนุมัติเพิ่มเติม

### 3.1 Report Invocation
