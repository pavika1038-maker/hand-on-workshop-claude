# Batch Functions

โฟลเดอร์นี้เก็บเอกสาร Functional Design สำหรับ **Batch Functions** (งานประมวลผลตามรอบเวลา)

## Naming Convention
- Prefix: `BAT`
- Format: `bat-[NNN]-[short-name].md`
- ตัวอย่าง: `bat-001-nightly-sync.md`, `bat-002-leave-balance-calc.md`

## สำคัญ
- ทุก function ต้องลงทะเบียนใน Function Index ก่อนสร้างเอกสาร
