## SRS Change Tracking

เมื่อได้รับการขอให้อัปเดต SRS document ให้ทำสิ่งต่อไปนี้เสมอ:
1. แก้ไขเนื้อหาตามที่ร้องขอ
2. เพิ่ม row ใหม่ใน Change Log section ของเอกสารนั้น
3. เพิ่ม row ใหม่ใน **10-requirement-definition/b0-system-requriement/srs-change-log.md** (central index)
4. แจ้งสรุปว่าเปลี่ยนอะไรบ้างพร้อม Change Type และ Source
ถ้าไม่ได้ระบุ Source ให้ถามก่อนบันทึก
